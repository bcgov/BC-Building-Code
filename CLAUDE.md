## Project Context

This repository contains the BC Building Code project, including XML-based building code documents, amendment generation tools, and JSON transformation pipelines.

## Important: Read Project Documentation First

Before working on this project, **read all documentation files under `.kiro/steering/`** to understand the project rules, structure, and guidelines:

- `product.md` - Product overview and requirements
- `tech.md` - Technical specifications and architecture
- `structure.md` - Repository structure and organization
- `BC Builiding Code - Amendment Creation Guide.md` - Guide for creating amendments
- `How to make Revision Amendments like Errata, Policy Change, Code revisions etc.md` - Revision amendment procedures
- `Global Text Replacements Feature.md` - Text replacement functionality
- `Fixing Amendment Validation Warnings.md` - Troubleshooting validation issues
- `repo_list_of_files.xml` - Complete file listing for the repository

## Quick Start

1. Read the steering documents in `.kiro/steering/`
2. Understand the XML schema and amendment structure
3. Use the JSON generation pipeline tools as documented
